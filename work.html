Relazione Progetto AWS - Sito Web Single Page

Per questo progetto, ho utilizzato Amazon Web Services (AWS) per creare un server virtuale in grado di ospitare un sito web single page. Il sito è stato containerizzato utilizzando Docker per gestire l'intero stack applicativo, compreso il server web, il database e PHP.

Ogni professore avrà delle credenziali personali.

per realizzare il sito ho seguito i seguenti passi:
- creazione istanza ec2 su aws - installazzione di docker sul server - creazione di due container, uno per il database (MariaDB) e uno per il web server (apache) - sostituzione del file di base di apache con il mio sito (via FTP) e creazione del DB

la repository GitHub contenente i file del sito web e altri materiali pertinenti può essere trovato qui.
