Relazione Progetto AWS - Sito Web Single Page

Per questo progetto, abbiamo utilizzato Amazon Web Services (AWS) per creare un server virtuale in grado di ospitare un sito web single page. Il sito è stato containerizzato utilizzando Docker per gestire l'intero stack applicativo, compreso il server web, il database e PHP.

Il sito web è stato sviluppato rispettando gli standard HTML5 e CSS3 per assicurare la compatibilità e l'accessibilità.

Per garantire la sicurezza dei dati sensibili, abbiamo predisposto credenziali separate per ciascun professore di sistemi che avrà accesso al sito.

Il repository GitHub contenente i file del sito web e altri materiali pertinenti può essere trovato qui.
